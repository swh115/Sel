Device Setup to Install Ask Helen App

Step 1: Factory Reset the Device

Note: If the device is new, you can skip this step.

  • Go to settings
  • Go to general management
  • Tap on reset
  • Tap on factory data reset

Step 2: Start the Android Setup

  • On the first screen of Android setup, tap the screen six times to open the QR code reader.

Step 2: Start the Android Setup (Part 2)

  • Go to CCC
  • Go to the "Device onboarding" section
  • Scan the universal enrollment QR code at the top left of the screen.

Note: This step registers the new device on Miradore, automatically including its serial number in the record.

Step 3: Complete the Android Setup

  • Follow the prompts to accept permissions and finish setting up the tablet.

Step 3: Complete the Android Setup (Part 2)

  • The app will automatically begin installing.

Step 4: Locate the Tablet's Serial Number

  • Find the serial number on the back of the tablet.

Step 4: Whitelist the Serial Number

  • Whitelist the serial number in the system.

Step 5: Generating Green QR Code in CCC

  • Go to the device section in CCC.
  • Search for the serial number you noted.

Step 5: Generating Green QR Code

  • Tap the QR Code button to generate a green QR code for the tablet.

Step 5: Scan the Green QR Code

  • Scan the green QR code with the tablet to complete the setup.

Step 6: Finalize the Setup

  • The app is now in recycled mode and ready for patient admission.
